# construcao

Trabalhando em uma construtora você começa a se deparar com problemas que muitas vezes não existem soluções personalizadas no mercado prontas para atender a demanda do cliente.

Pensando nisso, eu comecei a criar um projeto inicial onde seria possível organizar a gestão de uma construtora em python, utilizando json.

Ainda está na fase inicial, mas vou continuar atualizando o projeto sempre que possível.
